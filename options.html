Connect with Seth: Homepage | Twitter | Bluesky

How to Translate

To translate text on any webpage:

  1. Highlight the text you want to translate.
  2. Right-click on the highlighted text.
  3. Select the Translate to (language) with (provider) (UGTBrowser) option from the context menu.

Note: This option only shows up if you right-click highlighted text!

Comments or problems? Check out the UGTBrowser github.
